# botOfertas

Bot Telegram para monitorar grupos e encaminhar ofertas baseadas em alertas de palavras-chave.

## Funcionalidades

- Monitora múltiplos grupos simultaneamente
- Alertas personalizáveis por palavra-chave (ex: iphone, notebook)
- Encaminha mensagens automaticamente para chat destino
- Comandos de administração via grupo de controle:
  - /addgroup / /removegroup — gerenciar grupos monitorados
  - /addalert / /removealert — gerenciar alertas
  - /status — ver configuração atual
  - /search <termo> — buscar ofertas recentes
